## Lesson 1 Underwater World

### Key points:

* 1. When 'Go' button clicked (运行按钮被点击)  
* 2. Forever (永久循环)  
* 3. Wait ... seconds (等待...)  
* 4. If ... then ... (如果...那么...)  
* 5. If on edge, bounce (碰到边缘，反弹)  
* 6. Set rotation style (设置旋转方式)  
* 7. Set size to ... (设置大小)  
* 8. Switch costume to ... (更换造型)   
* 9. Play sound ... (播放声音)

### Steps:
1. Build a fish   
2. Make it move (with edge detection)   
3. Build more fish  
4. Build underwater background  
5. Add music
